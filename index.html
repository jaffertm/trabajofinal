Atajos de teclado para pestañas y ventanas

A la hora de operar con nuestro navegador es importante tener el control de las pestañas y ventanas.
Estamos tan acostumbrados a trabajar casi siempre de la misma manera con el ordenador que prácticamente lo hacemos todo de forma mecánica. Por lo general esto es bueno, salvo cuando por costumbre nos complicamos demasiado haciendo algunas cosas que podríamos acelerar con una simple combinación de teclas, por eso hoy os vamos a ofrecer una pequeña colección de atajos de teclado para Chrome.
Teclado de funciones para Chrome Teclas de páginas web Tecla Windows
Abrir el menú de Chrome -> Alt + F o Alt + E o F10 Abrir las opciones para imprimir la página actual -> Ctrl + P Abrir el Centro de actividades -> Win + A
Mostrar u ocultar la barra de marcadores -> Ctrl + Mayús + B Abrir las opciones para guardar la página actual -> Ctrl + S Llevar el foco al área de notificación -> Win + B
Abrir la página Historial en una pestaña nueva -> Ctrl + H Volver a cargar la página actual -> F5 o Ctrl + R Abrir el menú de botones de acceso -> Win + Mayús + C
Abrir el administrador de tareas de Chrome -> Mayús + Esc Detener una página que se está cargando -> Esc Mostrar y ocultar el escritorio -> Win + D
Abrir la barra de búsqueda para buscar en la página actual -> Ctrl + F o F3 Mostrar el código fuente HTML que no se puede editar de la página actual -> Ctrl + U Abrir Explorador de archivos -> Win + E
